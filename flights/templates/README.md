# Flight templates (reference only)

`a330-rr-base.FLT` is a **reference** dump from a real MSFS A330 autosave (titles/liveries).

**Do not** use it for mid-session `FlightLoad` from Challenge Lab — loading foreign full-session FLTs while already in free flight has crashed MSFS 2024.

Runtime Start Challenge uses **SimConnect teleport + time only** after verifying the correct aircraft is already loaded.
